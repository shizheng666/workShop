# HVG Workshop Microsite

HVG is a Next.js microsite for the ICPR 2026 workshop **Human-Centric Video Generation**.

## Features

- Multi-route workshop website with dedicated pages for home, schedule, submission, and committee
- English-facing UI with centralized editable workshop content
- CMT-first submission guidance and pre-submission policy display
- Legacy Google Form compatibility helpers retained for fallback workflows
- Vitest test suite for route rendering, content config, and registration helpers

## Development

```bash
npm install
npm run dev
```

## Verification

```bash
npm test
npm run lint
npm run build
```
